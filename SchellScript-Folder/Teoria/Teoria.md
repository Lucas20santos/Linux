# Shell Script

## O que é o Bash

Bash, abreviação de Bourne Again Shell, é um interpretador de shell de linha de comando e linguagem de script de código aberto, que serve como interface entre o usuário e o Sistema Operacional.

> Você pode usar o Bash de duas maneiras principais:

- Modo interativo: Também chamado de interpretador de comandos, permite a execução de comandos no terminal. Ele executa os comandos sequencialmente, caso haja múltiplos comandos.
- Script: Refere-se ao modo não interativo, que permite escrever sintaxe Bash contendo uma sequência de múltiplos comandos, executando-os um após o outro em um único arquivo.

> Características principais:

- Interativo: Insira comandos no terminal e execute-os em tempo real.
- Programação de scripts: Escreva vários comandos ou programas em um arquivo, execute-os e automatize as tarefas.
- Multiplataforma: Funciona em todas as plataformas, como Linux, MacOS, Windows e sistemas operacionais Unix.
- Compatibilidade com versões anteriores: As versões mais recentes do bash são compatíveis com versões mais antigas de comandos e scripts.
